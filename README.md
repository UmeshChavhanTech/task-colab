#  Task Collaboration Platform

A real-time task collaboration platform (Trello/Notion inspired) built using the MERN stack with WebSocket support.

---

## Features

- User Authentication (Register/Login)
- Create Boards
- Create Lists inside Boards
- Create / Update / Delete Tasks
- Drag & Drop Tasks
- Assign Users to Tasks
- Real-time updates using Socket.io
- Protected Routes
- Activity Tracking
- RESTful APIs
- MongoDB Atlas Integration

---

##  Tech Stack

### Frontend
- React.js
- Redux Toolkit
- React Router
- Material UI
- Axios

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT Authentication
- Socket.io
